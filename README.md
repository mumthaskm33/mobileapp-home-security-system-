# Project Overview
This project is a comprehensive security system that integrates facial recognition and behavioral analysis. It includes a Django backend for managing authorized users and tracking security logs, and a Flask application for serving the dashboard and handling real-time alerts.

## Key Features
- **Facial Recognition:** Detects and verifies authorized users.
- **Intruder Detection:** Logs and alerts on unrecognized faces.
- **Behavior Analysis:** Analyzes head pose and facial cues to detect suspicious behavior.
- **Dashboard:** Web interface for monitoring logs, adding authorized users, and viewing live feeds.

## Technologies Used
- **Language:** Python
- **Web Frameworks:** Flask, Django
- **Computer Vision:** OpenCV, MediaPipe
- **Database:** MySQL / SQLite
- **Frontend:** HTML, CSS

## Setup
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the Django backend:
   ```bash
   cd django_backend
   python manage.py runserver
   ```
3. Run the Flask application:
   ```bash
   python app.py
   ```
